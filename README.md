# 📚 LibRazor - Kütüphane ve Kitap Yönetim Sistemi
  
**ASP.NET Core Razor Pages** ve **ADO.NET** kullanılarak geliştirilmiş, modern ve dinamik kütüphane yönetim platformu.

</div>


## 💡 Proje Hakkında

**LibRazor**, kitaplarınızı ve yazarlarınızı kolayca yönetebileceğiniz, aynı zamanda sistemdeki verileri dışa aktarabileceğiniz kapsamlı bir web uygulamasıdır. Proje, veri erişim katmanında doğrudan **ADO.NET** kullanarak yüksek performanslı SQL sorguları yürütmek üzere tasarlanmıştır.

## ✨ Özellikler

- 🔐 **Kimlik Doğrulama:** Cookie tabanlı güvenli giriş (Login), kayıt olma (Register) ve oturum yönetimi.
- 📖 **Kitap ve Yazar Modülü:** ADO.NET altyapısı ile hızlı CRUD (Ekleme, Okuma, Güncelleme, Silme) operasyonları.
- 🔍 **Dinamik Arama:** Kayıtlar içerisinde hızlı filtreleme.
- 📊 **Gelişmiş Dışa Aktarma (Export):**
  - Tek tıkla verileri **Excel (.xlsx)** formatında indirme *(ClosedXML ile)*.
  - Verileri profesyonel görünümlü **PDF** formatında indirme *(QuestPDF ile)*.
- 🎨 **Modern Arayüz:** Bootstrap 5 (Stylish Portfolio) ile tam duyarlı (responsive) şık tasarım.

## 🛠 Teknolojiler

Projenin geliştirilme sürecinde aşağıdaki modern teknolojiler ve kütüphaneler kullanılmıştır:

| Kategori | Teknoloji / Kütüphane |
|---|---|
| **Backend** | `C#`, `ASP.NET Core Razor Pages (.NET 8)` |
| **Veritabanı** | `MS SQL Server` (LocalDB) |
| **Veri Erişimi** | `ADO.NET` (`Microsoft.Data.SqlClient`) |
| **Frontend** | `HTML5`, `CSS3`, `Bootstrap 5`, `Font Awesome` |
| **Araçlar / Kütüphaneler** | `ClosedXML` (Excel), `QuestPDF` (PDF) |
